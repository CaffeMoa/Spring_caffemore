# Spring_caffemore

- 스프링 서버용 (카카오맵 api 활용)

-**실행 영상(누르면 이동합니다)**

[![video](https://img.youtube.com/vi/mACT4j8VCHY/0.jpg)](https://youtu.be/mACT4j8VCHY?t=0s)

### 기능

- 주소 선택 시 (강남구 -> 강동구) 바뀐 주소대로 다시 카페 검색

- 지도 움직일 시 바뀐 위치 기반으로 카페 검색

- 왼쪽 리스트에 마우스 올릴 시 해당 카페위치에 카페 이름 나타남
